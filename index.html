SiteSeer: Visually Impaired Navigator
Raspberry Pi • Python • Google Cloud www.nyu.edu

Course Engineering Project • SiteSeer is a cost-effective application that improves the accessibility of travel for the visually impaired to navigate in the urban environment. Those who are visually impaired will find it difficult to complete what may be perceived as simple tasks by those who are not disabled such as crossing streets, finding directions, and safely reaching one’s destination. This project directly addresses the demand for more disability-friendly applications for the blind. • Configured Google Cloud & Camera/Sensors to operate on Raspberry Pi to provide feedback such as using image recognition and learning by utilizing Google APIs to provide directions and write an algorithm • Created iOS App: to obtain GPS location from the phone, provide live-feed, speech services (speech-to-text to get destination from user, text-to-speech to give user instructions)
